# ShortVideo

## 布局思考
1. 顶部设计一个组件，包含 Logo、搜索、发布作品、登录、用户信息 。这个组件在全局都不变的
2. 下方设计是左右布局，这个布局比例基本也不会变化。所以弄一个容器组件，左侧是列表，放一些按钮，右侧是 一个大方块，放视频或个人中心、上传控件等
3. 左侧可以放一些组件，右侧也是放一些组件。比较难的就是 数据流向 方面的了，之后再去思考这个。
